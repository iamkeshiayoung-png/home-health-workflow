# Home Health Visit Workflow

## Problem
Home health clinicians spend too much time documenting visits due to fragmented EHR workflows.

## Users
- Home Health Clinicians
- Clinical Managers
- QA Review Teams

## Goal
Reduce documentation time and improve compliance accuracy.

## Scope
This product focuses on improving visit documentation from start to completion.
## MVP Scope
- Start a visit
- Resume a visit
- Auto-save documentation
- Submit visit for QA
Commit to: mvp-scope
